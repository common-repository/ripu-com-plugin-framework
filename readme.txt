Plugin Framework für alle RIPU Com Plugins

== Description ==

Das RIPU Com Plugin Framework ist eine kleine Ansammlung von Klassen, die für die Benutzung von RIPU Com Wordpress Plugins von Nöten sind.

Mehr Informationen zum Plugin findet man auf der [RIPU Com Webseite](http://www.ripucom.de/) oder direkt auf der Seite [RIPU Com Wordpress Plugin Framework](http://www.ripucom.de/opensource/ripu-com-plugin-framework)

Thanks to famfamfam for the icons!

== Installation ==

Das Framework wird wie ein ganz normales Wordpress Plugin in den Ordner

/wp-content/plugins/

Entpackt und aus dem Adminmenü von Wordpress aus aktiviert.

== Frequently Asked Questions ==

= Darf ich das Framework auch für eigene Plugins verwenden? =

Natürlich! Ich bitte darum, mich bei Benutzung mit einer kleinen eMail zu informieren. Ich werde dann auch auf meiner Webseite über das neu entwickelte Plugin berichten!
